智慧迭代器 (Wisdom Iteration Engine)

by Adam Chan. 2024-09 (v0.3) 如果這工具對你有用，歡迎發個訊息給作者以示鼓勵。
注意︰本程序使用 {{ MODEL_NAME }} 的內置知識庫回答問題 (數據更新至 2023 年 10 月)，沒有提供網上搜索功能，因此沒有實時數據。

這個程序是一個「多層次的答案系統」，旨在通過 LLM 迭代優化來生成高質量的回答，通過多次評估和優化，不斷改進初始回答，最終得到一個更全面、準確、深入的答案。這個系統特別適用於需要深度分析和全面解答的複雜問題。以下是主要功能、技術參數和運作原理：

  • 輸入問題 (user_question)： 用戶輸入的問題將成為生成答案的基礎。
  • 直接 LLM 回答 (Direct LLM Answer)： 使用 LLM 生成的直接回答。
  • 多層次 LLM 回答 (Final Optimized LLM Answer)： 通過多輪優化和評估來生成最終的優化答案。
  • 比較結果 (Comparison Result)： 比較直接 LLM 回答與多層次 LLM 回答的質量、全面性和準確性。
  • 日誌 (Logs)： 顯示多層次 LLM 回答過程中的操作日誌。

技術參數：

  • LLM 模型名稱： {{ MODEL_NAME }}
  • 最大 token 數量 (max_tokens)： 2000
  • 溫度設定 (temperature)： 初始回答 0.5，直接回答 0.7，優化詢問 0.7
  • 最大迭代次數 (max_iterations)： 3

運作原理：

本工具基於以下步驟運作：

  1. 生成初始回答： 根據用戶的問題生成初始回答，並計算使用的 token 數量。
  2. 進行評估： 使用評估提示語來評估初始回答的質量，並為每個方面打分（準確性、全面性、深度、相關例子的使用、論證的邏輯性）。
  3. 優化回答： 根據評估的結果生成新的優化提示語，進行多輪迭代優化，直至滿足設置的評分閾值或達到最大迭代次數。
  4. 最終回答： 選擇評分最高的回答作為最終結果，並將操作日誌記錄下來。
  5. 答案比較： 使用更高級的 LLM 來比較直接 LLM 回答和多層次 LLM 系統的答案，從質量、全面性和準確性三方面進行評分。
